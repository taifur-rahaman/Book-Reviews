# Book Review Template

## Default Template Structure

When creating a new book review, use the following structure:

```markdown
# [Title]

![Cover Image](../utils/covers/[book_name].png)

| | |
|---|---|
| Original Title | [Original language title, or same if English] |
| Alternative Title | [Alternative titles, or -] |
| Publication Year | [Year] |
| Author | [Author name] |
| Publisher | [Publisher name] |
| Language | Original: [Language] / Read In: [Language] |
| Genre | [Genres] |
| Format | Paperback / Hardcover / eBook / Audiobook |
| Status | Reading / Completed / On Hold / Dropped |
| Pages Read | [X] / [Total] |
| Start Date | [Date] |
| Last Read | [Date] |
| End Date | [Date or -] |
| Rating | [Rating] / 10 |

## Overview

[Brief overview of the book and its premise]

## Story & World

[Details about the plot and world-building / setting]

## Characters

[Character descriptions and development]

## Writing Style & Prose

[Notes on the author's writing style, prose quality, narrative voice, pacing]

*Rating: [X]/10*

## Themes & Impact

- **Theme 1**: Description
- **Theme 2**: Description

## Personal Notes & Observations

[Personal thoughts and observations]

## Reread Value

**Would I reread?** [Yes/No/Maybe]

- Book 1 (Author - Description)

```

## Important Notes

1. **Cover Images**: ALWAYS generate and include a cover image at the top of each review
2. **Metadata**: Ensure all metadata fields are filled accurately
3. **Format**: Specify the format read (eBook, Paperback, Hardcover, Audiobook)
4. **Language**: Note both the original language and the language you read it in
5. **Rating Scale**: Use the 10-point scale with detailed breakdown
6. **Update Files**: Always update README.md and temporal_distribution.md when adding new books

## Cover Image Generation

When generating cover images, use prompts that:
- Match the book's tone and genre
- Include the title and author in the image
- Reflect key themes, settings, or imagery from the book
- Use atmospheric, evocative colors suited to the genre
- Feel like a professional book cover design
